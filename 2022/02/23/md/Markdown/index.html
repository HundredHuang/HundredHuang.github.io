Markdown基本语法

Hundred2022-02-232025-02-17

一、标题

    使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

二、段落样式

段落换行：用两个以上空格加回车

字体：

分割线：用三个以上的（星号）、-（减号）、_ （下划线）建立分割线，行内不能有其他内容，可在号或-号中间加入空格

删除线：在文字的两端加上两个波浪线即可

下划线：通过 HTML 的 标签来实现

脚注：[^我是脚注]

三、列表

有序列表：使用 数字加 . 号加空格表示

无序列表：使用星号(**)、加号(*+)或减号(-) 加空格表示

嵌套列表：需在子列表中的选项前面添加四个空格

四、区块

区块引用：在段落开头使用 > 符号加空格

区块嵌套：一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推

区块中使用列表

列表中使用区块

五、代码块

段落中的函数：函数前后使用反引号 我是函数包起来

代码区块：用 4 个空格、一个制表符（Tab 键）或3个反引号表示

六、链接

普通链接：[链接名称](链接地址) 或 <链接地址>

高级链接：通过变量设置一个链接，变量在文档末尾赋值

七、图片

语法格式：![alt 属性文本](图片地址)或![alt 属性文本](图片地址 "可选标题")

图片使用变量

使用本地图片

八、表格

语法：使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行

九、高级技巧

支持的HTML元素：目前支持的 HTML 元素有

转义： 使用反斜杠转义特殊字符
